# Adjust style/labels for an `er_plot` object

Not yet implemented – currently a no-op placeholder for future styling
customisation (labels, theme, formatters). See `object$style` (set by
[`er_plot()`](https://erplots.djnavarro.net/reference/er_plot.md)) for
what's already there to be made adjustable.

## Usage

``` r
er_plot_style(object, labels)
```

## Arguments

- object:

  Partially constructed plot (has S3 class `er_plot`)

- labels:

  Named list of labels

## Value

The input `object`, unchanged

[`er_plot()`](https://erplots.djnavarro.net/reference/er_plot.md)
